Pleasure Ridge Park High School

Pleasure Ridge Park High School worked with me to redesign their website to present visitors with an online experience that allows for the PRP community to access information quickly and easily. Pleasure Ridge Park High School wanted users to be able to, view availble courses, athletics, contact staff and special events while providing users quick access to daily announcements and counslors newsletters.

Features

Responsive

Using modern design techniques such as Flex box layouts to scale the site to look and function well on any screen size from the smallest phone to a large computer monitor.

Announcements

To pull the latest announcement from Ridge Talk and display them in a clean design, I took advantage of the Blogger JSON API to accomplish this task.
